# ARCH-001: Modelo de Contexto del Sistema (C4 - Nivel 1)

## 1. Declaración de Propósito
[cite_start]El Project Knowledge Base (PKB) es el núcleo centralizado que transforma el conocimiento informal y fragmentado del dominio MIPSP en activos de ingeniería semánticos, estructurados y ejecutables[cite: 4, 161].

## 2. Actores y Límites del Sistema
* [cite_start]**Knowledge Engineer (Human):** Interactúa mediante la CLI para gobernar, validar e ingerir lotes de conocimiento [cite: 254-256].
* [cite_start]**Repository (File System):** Almacenamiento local e inmutable indexado mediante Git [cite: 123-136].
* [cite_start]**MIPSP Legacy Sources:** Archivos Markdown origen conteniendo el histórico de especificaciones de las sesiones de diseño[cite: 160, 416].
