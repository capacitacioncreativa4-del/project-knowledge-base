# Proyecto 1 — Kit de Capacitación para Personal Operativo de Seguridad Privada

## Alcance de esta carpeta

Esta carpeta contiene **exclusivamente** el Proyecto 1 original: el kit de capacitación para
personal operativo (guardias) de empresas de seguridad privada, alineado a la normatividad
federal y del Estado de Morelos, y validado contra el Estándar de Competencia EC0060.

**Esta carpeta NO contiene, y no debe mezclarse con:**
- La Norma Editorial del Modelo Integral para la Profesionalización de la Seguridad Privada (MIPSP).
- El Kit Editorial Maestro (MIPSP Editor).
- El Project Knowledge Base (PKB) institucional, ubicado en `projects/mipsp/` de este repositorio.

Esta separación es intencional: el Proyecto 1 se desvió en su momento hacia esos otros
subproyectos antes de completarse. Esta carpeta existe para que el kit de capacitación
pueda avanzar y cerrarse de forma independiente.

## Contenido

- **`00-Indice-General-del-Kit.md`** — Portada y mapa del kit completo: tabla maestra de los
  21 cursos, mapa de dependencias, cursos de criticidad "Muy Alta", cobertura normativa y
  pendientes de decisión.

- **`01-fuentes-originales/`** — Los 47 archivos originales del Lote 1
  (`MIPSP-CONV-0001-P01.md` a `P47.md`), que contienen el análisis normativo (federal, estatal
  y EC0060) del cual se derivó el catálogo de cursos. Se conservan tal cual, sin editar, como
  respaldo y trazabilidad del origen del proyecto.

- **`02-kit-de-cursos/`** — Los 21 paquetes documentales completos (uno por curso), cada uno
  con sus 17 documentos: programa institucional, carta descriptiva, guía del instructor,
  manual del participante, presentación, cuaderno de ejercicios, banco de casos, manual de
  simulaciones, prácticas en campo, guía de roles, banco de reactivos, exámenes, rúbrica,
  lista de cotejo, guía de observación, evaluación de desempeño y PNO.
