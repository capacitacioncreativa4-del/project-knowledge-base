# Registro de Prefijos Oficiales del PKB (v1.0.0)

Garantiza la unicidad y el ruteo semántico del grafo de conocimiento[cite: 12]:

* **CONV**: Conversation (Fuentes primarias de origen)[cite: 12]
* **KP**: Knowledge Package (Contenedores lógicos de distribución)[cite: 12]
* **SES**: Engineering Session (Bloques de trabajo de diseño)[cite: 12]
* **ADR**: Architecture Decision Record (Puntos de inflexión técnica)[cite: 12]
* **REQ**: Requirement (Obligaciones funcionales y operativas)[cite: 12]
* **STD**: Standard (Reglas, procesos y políticas uniformes)[cite: 12]
* **ART**: Conceptual Artifact (Modelos y marcos abstractos)[cite: 12]
* **DOC**: Editorial Document (Productos listos para publicación)[cite: 12]
* **SYS**: Institutional System (Arquitectura empresarial viva)[cite: 12]
