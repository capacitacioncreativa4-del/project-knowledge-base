* **Release 0.3.0:** Knowledge Graph (Generación de enlaces y base relacional).

* **Release 0.5.0:** Command Line Interface (CLI de control local).
* **Release 1.0.0:** GA (General Availability) - Producto PKB Estable y listo para producción multiproyecto.
