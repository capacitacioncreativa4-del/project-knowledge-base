# Definition of Done (DoD) — PKB Platform

[cite_start]Un incremento se considera formalmente "Terminado" y listo para su integración en la rama `develop` solo si cumple con los siguientes criterios obligatorios[cite: 67]:

1. [cite_start]**Validación Sintáctica:** Todos los archivos de configuración y datos YAML cumplen al 100% con sus esquemas JSON correspondientes[cite: 70].
2. [cite_start]**Trazabilidad de Fuentes:** Los objetos de dominio (`projects/`) referencian explícitamente el ID de la conversación de origen en Markdown[cite: 73].
3. **Registro de Cambios:** La versión y modificaciones del incremento han sido asentadas con precisión en el `CHANGELOG.md` general.
