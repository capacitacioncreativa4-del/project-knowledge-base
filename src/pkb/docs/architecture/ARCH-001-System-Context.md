# ARCH-001: System Context Model

## 1. Introducción
El Project Knowledge Base (PKB) es un motor que procesa, valida y estructura el conocimiento de ingeniería organizativa de la suite MIPSP[cite: 2].

## 2. Diagrama de Bloques de Contexto
* **Usuario (Knowledge Engineer):** Interactúa mediante la interfaz de línea de comandos (CLI) para ingerir, validar y consultar activos.
* **Fuentes Primarias (Sources):** Conversaciones históricas (`.md`) que sirven de materia prima inmutable.
* **Base de Datos Semántica (Target):** Paquetes de Conocimiento estructurados (`.yaml`) listos para ser consumidos por humanos y automatizaciones.
