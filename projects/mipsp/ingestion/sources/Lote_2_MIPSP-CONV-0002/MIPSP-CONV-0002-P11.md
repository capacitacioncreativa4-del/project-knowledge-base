## Compromiso editorial

A partir de ese momento, cada archivo se desarrollará con criterios de producción profesional:

* **Control documental** (SICD-MIPSP, CCD-MIPSP y SIGD-MIPSP).
* **Estructura editorial uniforme** para toda la colección.
* **Referencias cruzadas** entre documentos.
* **Trazabilidad normativa** hasta el artículo, disposición o estándar de competencia correspondiente.
* **Preparación para impresión y distribución digital**.
* **Compatibilidad con Microsoft Office**, evitando funciones experimentales o dependencias innecesarias.
* **Control de versiones**, de manera que ningún documento tenga que rehacerse posteriormente.

## Una mejora que incorporaremos desde el primer archivo

Quisiera añadir un elemento que, en mi experiencia, diferencia una biblioteca documental convencional de un verdadero sistema institucional.

Cada documento incluirá un **Cuadro de Metadatos Institucionales**, por ejemplo:

| Campo                   | Valor                  |
| ----------------------- | ---------------------- |
| Código documental       | CUR-OPS-001-D01        |
| Serie                   | Operaciones            |
| Curso                   | Control de Accesos     |
| Documento               | Programa Institucional |
| Versión                 | 1.0                    |
| Estado                  | Vigente                |
| Propietario             | Dirección Académica    |
| Clasificación           | Interno                |
| Fecha de emisión        |                        |
| Próxima revisión        |                        |
| Documentos relacionados | SICD                   |
| Baseline                | BL-1                   |

Estos metadatos facilitarán la gestión documental, las auditorías, la automatización y la futura migración a un gestor documental (DMS/ECM) si el proyecto evoluciona hacia una plataforma institucional.

## Mi compromiso adicional

Quiero asumir un compromiso más ambicioso con este proyecto.

No me limitaré a generar documentos técnicamente correctos. Procuraré que la colección tenga un nivel de calidad comparable al de los manuales utilizados por organismos de certificación, instituciones de educación superior y centros de formación corporativa.

Eso implica que, además del contenido técnico, cuidaré aspectos como:

* coherencia terminológica en toda la colección;
* uniformidad gráfica y editorial;
* consistencia entre objetivos, actividades y evaluación;
* alineación permanente con la normativa fuente;
* facilidad de actualización mediante control de configuración;
* preparación para procesos de auditoría y acreditación.

Después de la cantidad de trabajo que ya hemos invertido, considero que el proyecto merece ese nivel de rigor.

Cuando la herramienta de generación de documentos esté disponible, no comenzaremos con prototipos: comenzaremos directamente con la **versión institucional** del **Kit Editorial Maestro**, que será la base permanente sobre la que construiremos toda la biblioteca del MIPSP.
