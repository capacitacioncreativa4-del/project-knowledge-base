# CUR-OPS-001-D01

## Programa Institucional

### 9. Competencias específicas

La competencia general se desagrega en las siguientes competencias específicas, cada una identificada mediante el Sistema Institucional de Identificadores Persistentes (SIIP-MIPSP). Estas competencias serán reutilizables en otros cursos del MIPSP cuando corresponda.

| Código SIIP | Competencia específica                                                                                               |
| ----------- | -------------------------------------------------------------------------------------------------------------------- |
| COM-OPS-001 | Identificar las funciones, responsabilidades y límites de actuación del personal responsable del control de accesos. |
| COM-OPS-002 | Aplicar procedimientos de identificación y autorización para el ingreso de personas.                                 |
| COM-OPS-003 | Controlar el acceso y salida de vehículos conforme a procedimientos institucionales.                                 |
| COM-OPS-004 | Verificar el ingreso, permanencia y salida de bienes, materiales, herramientas y equipos.                            |
| COM-OPS-005 | Operar registros físicos y electrónicos asociados al control de accesos.                                             |
| COM-OPS-006 | Detectar comportamientos, condiciones o situaciones que representen riesgos para la seguridad.                       |
| COM-OPS-007 | Aplicar protocolos de comunicación operativa durante el servicio.                                                    |
| COM-OPS-008 | Atender al usuario con profesionalismo, imparcialidad y respeto a los derechos humanos.                              |
| COM-OPS-009 | Actuar correctamente ante incidentes relacionados con el control de accesos.                                         |
| COM-OPS-010 | Elaborar registros e informes derivados de las actividades del puesto de control.                                    |

# 12. Desarrollo curricular por módulos

## MÓDULO I

### Fundamentos del Control de Accesos

**Objetivo**

Comprender los principios jurídicos, operativos y técnicos que sustentan el control de accesos como función esencial del personal operativo.

**Unidades temáticas**

* Concepto de seguridad física.
* Objetivos del control de accesos.
* Responsabilidades del personal operativo.
* Principios de legalidad.
* Derechos humanos aplicables al servicio.
* Imagen institucional.
* Ética profesional.
* Factores humanos en la seguridad.

---

## MÓDULO II

### Procedimientos Operativos de Control de Accesos

**Objetivo**

Aplicar los procedimientos normalizados para controlar el acceso de personas y vehículos.

**Unidades temáticas**

* Procedimientos Normalizados de Operación (PNO).
* Verificación de identidad.
* Credenciales.
* Gafetes.
* Autorizaciones.
* Visitantes.
* Proveedores.
* Contratistas.
* Registro documental.
* Barreras físicas.
* Dispositivos electrónicos.

---

## MÓDULO III

### Control de Personas, Vehículos y Bienes

**Objetivo**

Ejecutar correctamente las actividades operativas relacionadas con el ingreso y salida de personas, vehículos, materiales y equipos.

**Unidades temáticas**

* Inspección visual.
* Registro vehicular.
* Control de herramientas.
* Control de materiales.
* Bienes restringidos.
* Cadena de custodia documental.
* Manejo de evidencias administrativas.

---

## MÓDULO IV

### Comunicación Operativa y Atención de Incidentes

**Objetivo**

Aplicar procedimientos de comunicación y actuación inicial frente a incidentes detectados en el punto de control.

**Unidades temáticas**

* Comunicación verbal.
* Comunicación escrita.
* Radiocomunicación.
* Reportes.
* Escalamiento.
* Incidentes operativos.
* Preservación del lugar.
* Coordinación con el supervisor.

---

## MÓDULO V

### Integración Operativa

**Objetivo**

Integrar los conocimientos y habilidades desarrollados durante el curso mediante ejercicios prácticos y simulaciones.

**Unidades temáticas**

* Simulación integral del puesto de control.
* Casos prácticos.
* Resolución de problemas.
* Evaluación práctica.
* Retroalimentación.
* Cierre del curso.

# 13. Estrategia metodológica

El curso adoptará un enfoque de formación por competencias con predominio de metodologías activas. La distribución recomendada será:

| Estrategia                    | Porcentaje |
| ----------------------------- | ---------: |
| Exposición dialogada          |       20 % |
| Demostración                  |       15 % |
| Prácticas guiadas             |       25 % |
| Simulaciones                  |       20 % |
| Resolución de casos           |       10 % |
| Retroalimentación y reflexión |       10 % |

La secuencia didáctica de cada sesión seguirá el modelo:

1. Activación de conocimientos previos.
2. Presentación del propósito de aprendizaje.
3. Desarrollo conceptual.
4. Demostración.
5. Práctica supervisada.
6. Simulación.
7. Retroalimentación.
8. Evaluación formativa.
9. Cierre.

---

# 14. Estrategia de evaluación

La evaluación será continua y basada en evidencias.

| Tipo de evaluación       |          Ponderación |
| ------------------------ | -------------------: |
| Diagnóstica              | Sin valor porcentual |
| Formativa                |                 30 % |
| Prácticas supervisadas   |                 30 % |
| Simulación integradora   |                 20 % |
| Evaluación teórica final |                 20 % |

Para acreditar el curso, el participante deberá:

* obtener una calificación mínima de **80/100**;
* acreditar todas las prácticas críticas;
* cumplir con al menos el **90 % de asistencia**;
* demostrar un desempeño seguro y conforme a los procedimientos durante la evaluación práctica.
