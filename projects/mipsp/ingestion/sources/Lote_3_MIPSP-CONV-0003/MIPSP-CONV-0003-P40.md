La arquitectura actual ya ha superado el concepto tradicional de un sistema documental. En este punto el ecosistema es capaz de:

* Capturar conocimiento.
* Gestionar procesos.
* Ejecutar automatizaciones.
* Analizar información.
* Capacitar personas.
* Modelar operaciones.
* Generar inteligencia.

El siguiente paso es utilizar todo ese conocimiento acumulado para apoyar la toma de decisiones de alto nivel.

Una organización madura no solamente necesita saber:

> "¿Qué está ocurriendo?"

También necesita responder:

> "¿Qué puede ocurrir si tomamos una decisión determinada?"

y:

> "¿Cuál es la mejor alternativa considerando riesgos, recursos y objetivos?"

Para ello se incorpora:

# MIPSP-Editor

# SUBSYSTEM-14 — Strategic Intelligence, Scenario Planning & Decision Support Layer (SISPDSL)

---

# Objetivo

Crear una capa de inteligencia estratégica capaz de apoyar decisiones institucionales mediante:

* Análisis prospectivo.
* Simulación de escenarios.
* Evaluación de alternativas.
* Modelos de impacto.
* Planeación estratégica.
* Recomendaciones basadas en evidencia.
