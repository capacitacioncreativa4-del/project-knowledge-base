# Fase IV — Biblioteca Institucional MIPSP

Finalmente, todos los cursos se integrarán en una biblioteca con un índice maestro, referencias cruzadas y un sistema de codificación homogéneo.

## Estándares editoriales

Como siguiente paso propongo formalizar un conjunto de normas que aplicarán a todos los documentos de la colección. Entre ellas:

| Elemento           | Estándar propuesto                             |
| ------------------ | ---------------------------------------------- |
| Tamaño de papel    | Carta (principal) y A4 (edición internacional) |
| Márgenes           | Uniformes para impresión a doble cara          |
| Tipografía         | Familia única para cuerpo, títulos y tablas    |
| Numeración         | Decimal multinivel (1, 1.1, 1.1.1...)          |
| Figuras            | Numeración automática por capítulo             |
| Tablas             | Numeración automática por capítulo             |
| Encabezados        | Código documental, versión y clasificación     |
| Pie de página      | Número de página y estado del documento        |
| Control de cambios | Historial obligatorio en todos los documentos  |
| Referencias        | Sistema uniforme en toda la colección          |

## Incorporación de un Índice Maestro de Referencias (IMR)

Propongo añadir un componente que reforzará notablemente la consistencia de la biblioteca:

**IMR-MIPSP — Índice Maestro de Referencias**

Este índice centralizará todas las relaciones entre documentos. Por ejemplo:

* qué ETS desarrollan cada competencia;
* qué formatos respaldan cada procedimiento;
* qué reactivos evalúan cada resultado de aprendizaje;
* qué diapositivas corresponden a cada sesión;
* qué anexos complementan cada documento.

Con ello, cualquier modificación futura podrá analizarse en términos de impacto documental antes de ser aprobada, fortaleciendo el **Control de Configuración Documental (CCD-MIPSP)** y el **Sistema Institucional de Gobernanza Documental (SIGD-MIPSP)**.
