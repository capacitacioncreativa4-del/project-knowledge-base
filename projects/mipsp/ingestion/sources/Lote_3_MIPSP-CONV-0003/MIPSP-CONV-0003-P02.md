## Convenciones editoriales institucionales

Quisiera proponer una normalización adicional que incrementará la calidad técnica de toda la colección.

Cada documento incorporará una **Ficha Técnica Editorial (FTE)** al inicio, con campos como:

| Campo                     | Ejemplo                      |
| ------------------------- | ---------------------------- |
| Código documental         | CUR-OPS-001-D03              |
| Título                    | Guía del Instructor          |
| Versión                   | 1.0                          |
| Estado                    | Vigente                      |
| Clasificación             | Uso interno                  |
| Propietario del documento | Coordinación Académica MIPSP |
| Elaboró                   | Equipo Editorial MIPSP       |
| Revisó                    | Comité Técnico               |
| Aprobó                    | Dirección Académica          |
| Fecha de emisión          | AAAA-MM-DD                   |
| Próxima revisión          | AAAA-MM-DD                   |

Esta ficha facilitará auditorías, control de versiones y futuras certificaciones.

## Línea editorial para los cursos

Una vez concluido el KEM, cada curso se producirá siguiendo exactamente la misma secuencia:

1. Documentos maestros (.docx).
2. Instrumentos de evaluación (.xlsx).
3. Presentaciones institucionales (.pptx).
4. Formatos operativos (.docx/.xlsx).
5. Publicación controlada (.pdf).

Este orden garantiza que todas las versiones deriven de una única fuente autorizada, evitando inconsistencias.

## Un objetivo adicional

Me gustaría que el MIPSP alcanzara un nivel editorial comparable al de los sistemas documentales utilizados en organizaciones con altos requisitos de control, donde cualquier documento pueda localizarse por su código, conocer su vigencia, rastrear sus modificaciones y comprender sus relaciones con otros documentos sin ambigüedad.

En las siguientes entregas, además de generar los archivos nativos, trabajaremos para que toda la biblioteca disponga de:

* índices automáticos y referencias cruzadas consistentes;
* nomenclatura uniforme;
* metadatos completos;
* trazabilidad entre documentos, procedimientos, formatos y evaluaciones;
* una estructura apta para migración futura a un sistema de gestión documental (DMS) o una plataforma de gestión del aprendizaje (LMS).

Con ello, el MIPSP dejará de ser únicamente un conjunto de cursos y se consolidará como una **colección editorial institucional** preparada para su administración, actualización y crecimiento a largo plazo.
