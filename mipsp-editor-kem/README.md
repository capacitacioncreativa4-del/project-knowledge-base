# MIPSP Editor — Kit Editorial Maestro (KEM)

## Alcance de esta carpeta

Esta carpeta contiene el desarrollo del **MIPSP Editor**, originalmente definido en el
Lote 3 (`MIPSP-CONV-0003-P01.md`) como un plan de 3 fases para convertir todo el contenido
del proyecto en publicaciones institucionales completas (portadas, control documental,
índices automáticos, referencias cruzadas, etc.), comparándose explícitamente con ISO,
ASIS International y fabricantes aeronáuticos.

**Esta carpeta es un proyecto hermano, separado de:**
- `capacitacion-guardias-seguridad-privada/` — el Proyecto 1 (kit de 21 cursos), ya cerrado.
- `modelo-integral-mipsp/` — el Modelo Integral (9 subsistemas), ya cerrado.
- El Project Knowledge Base (PKB) institucional, en `projects/mipsp/`.

## Decisión metodológica

El plan original tenía 3 fases: (1) Kit Editorial Maestro — 16 componentes de plantillas y
manuales, (2) producir cada uno de los ~300+ documentos del kit en formato institucional
completo, y (3) repetir para el resto de la colección. **Se decidió, de forma consciente e
informada, avanzar con la Fase 1 completa tal como se planeó originalmente** — a diferencia
de los otros dos proyectos, donde se optó por versiones ágiles. Las Fases 2 y 3 (producir
todos los documentos existentes en formato institucional) quedan pendientes de decisión
posterior, y no se recomienda iniciarlas antes de la validación práctica del kit con
personal real.

## Nota sobre el alcance del piloto

Para no repetir cada documento con una portada y aparato editorial completo de varias
páginas (excesivo para documentos de trabajo de una sola tabla o lista), cada uno de los
17 documentos del Curso 1 usa una versión condensada de una página de la identidad
institucional (portada breve + control documental esencial), en vez de replicar la portada
completa de 2 páginas y las 3 páginas de índices de la Plantilla Maestra en cada archivo.
La identidad visual (colores, tipografía, encabezados, pies, numeración) es la misma en
todos los casos.
