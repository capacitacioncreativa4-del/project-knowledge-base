# Modelo de Ciclo de Vida Documental

## Propósito
Definir los estados oficiales, las transiciones permitidas y las reglas de aprobación que rigen a los activos del Project Knowledge Base (PKB). El objetivo es asegurar un flujo controlado de ingeniería de la información, garantizando que solo el conocimiento validado alcance el estado de producción.
